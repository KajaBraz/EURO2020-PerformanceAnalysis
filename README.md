# FootballScorersAnalysis
Uncommon statistics about football tournaments presented from different perspectives:
- clubs/leagues and the number of footballers who played for them during the season that preceeded the tournament
- goals/assists: by player, by country, by club, by league, by age, and by height

The presented data refers to the last two editions of EURO and Copa America tournaments (2020 and 2024).

See all the available stats: https://kajabraz.github.io/FootballCompetitionsStats/
